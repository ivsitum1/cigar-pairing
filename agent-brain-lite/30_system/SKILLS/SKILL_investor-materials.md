# Skill: Investor materials

## When to use

- Drafting or updating fundraising collateral with consistent numbers and narrative

## Procedure

1. **Story:** problem, solution, market, traction, team, ask.
2. **Numbers:** single source of truth table; reconcile deck, memo, model.
3. **Risks:** honest limitations section.
4. **Format:** slide outline or memo per user request; no fabricated metrics.

- [ ] Figures consistent across sections
- [ ] No unverifiable market size without source tag
