# Citation styles (manuscript-facing)

## Vancouver (numbered)

- Order of appearance in text.
- Journal abbreviations per target journal list.
- Verify DOI/PMID for critical references.

## APA (author–date)

- (Author, Year) in text; full reference list alphabetically.
- Et al. rules per edition.

## Workspace rules

- Never fabricate references.
- Protocol-only papers: cite design, not unpublished results as outcomes.
- Distinguish review article (publication type) from internal screening corpus in prose.

## Tools

Use reference manager export when user provides; do not invent PMIDs.
