# Overclaim Pattern Rules

## Core Rule
Look for common forms of evidentiary inflation.

## Common Patterns
- association written as causation
- supportive biology written as mechanism proof
- model performance written as clinical usefulness
- external validation written as universal generalizability
- translational interest written as implementation readiness
- exploratory signal written as robust established finding

## Important Rule
Overclaiming can happen even when the manuscript uses polished or cautious-sounding language.

## Reporting Rule
Name the inflation pattern explicitly when present.
