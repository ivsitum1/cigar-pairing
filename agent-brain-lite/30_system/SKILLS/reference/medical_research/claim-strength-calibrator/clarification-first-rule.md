# Clarification-First Rule

## Core Rule
Do not produce a long claim-strength calibration review if the relevant manuscript text and evidence context are still too incomplete.

## Ask Before Expanding
Ask focused follow-up questions first when any of the following are unclear:
- what manuscript text is being checked,
- what study design produced the claim,
- what evidence type underlies the wording,
- what the validation status is,
- whether the user wants a broad screen or a focused high-risk audit.

## Reporting Rule
If the input is insufficient, explicitly tell the user what information would improve claim-calibration accuracy.
