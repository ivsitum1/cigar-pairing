# Claim Rewrite Boundary Rules

## Core Rule
Claim calibration should preserve meaning while bringing wording back inside the true evidence boundary.

## Acceptable Actions
- soften the verb
- narrow the scope
- reduce generalizability
- separate support from proof
- move from action language to implication language
- retain strong wording only when the evidence truly supports it

## Important Rule
Do not replace one problem with another.
Over-softening can make the manuscript vague and unhelpful.

## Reporting Rule
State whether a claim should be softened, narrowed, re-anchored, or left unchanged.
