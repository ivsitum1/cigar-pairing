# Severity Classification Rules

## Core Rule
Reference-integrity issues must be classified by severity.

## Severity Levels
- Major integrity risk
- Moderate citation concern
- Minor alignment issue
- Uncertain due to missing source access

## Important Rule
A citation mismatch supporting a central claim is not equivalent to a minor over-broad background citation.

## Reporting Rule
Prioritize issues that could materially undermine reviewer trust in the manuscript.
