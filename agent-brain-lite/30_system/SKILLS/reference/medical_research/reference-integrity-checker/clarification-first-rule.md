# Clarification-First Rule

## Core Rule
Do not produce a long reference-integrity review if the relevant manuscript text and source material are still too incomplete.

## Ask Before Expanding
Ask focused follow-up questions first when any of the following are unclear:
- what manuscript text is being checked,
- which claims are high priority,
- which references are attached to those claims,
- whether source abstracts or full texts are available,
- whether the user wants a broad screen or a focused audit.

## Reporting Rule
If the input is insufficient, explicitly tell the user what information would improve reference-integrity accuracy.
