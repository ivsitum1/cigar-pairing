# Claim-Source Matching Rules

## Core Rule
A citation is only adequate if it supports the actual claim being made, not just the general topic.

## Check For
- same population or context
- same intervention/exposure/domain
- same evidence level
- same direction of conclusion
- same strength of inference

## Important Rule
Topical relevance is not enough.
A related paper is not automatically a valid supporting source.

## Reporting Rule
Make clear when the problem is full mismatch versus partial or weak alignment.
