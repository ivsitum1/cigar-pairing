# Logic Reporting Rule

## Core Rule
Every major integrity judgment should be explained.

## Explain In Terms Of
- claim-source mismatch,
- evidence-boundary overreach,
- source-type mismatch,
- credibility risk,
- reviewer criticism risk,
- and correction urgency.

## Important Rule
Do not say only that a citation is “weak.”
State why it is weak and what kind of correction it needs.
