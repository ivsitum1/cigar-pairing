# Second-Hand Citation Rules

## Core Rule
A cited paper should not be used as if it directly established a claim that it only mentioned second-hand or inherited from another source.

## Common Warning Signs
- review articles cited as if they were primary data
- primary claims attributed to papers that only summarize prior work
- consensus or guideline claims attached to secondary literature without direct confirmation
- citation chains where the quoted statement is not traceable to the cited source

## Important Rule
Second-hand citation risk is a credibility problem even when the topic area is correct.

## Reporting Rule
State when the citation should be replaced with a primary source or directly verified.
