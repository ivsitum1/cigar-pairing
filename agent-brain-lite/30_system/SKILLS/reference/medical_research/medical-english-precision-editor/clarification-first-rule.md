# Clarification-First Rule

## Core Rule
Do not produce a long precision edit if the text and its scientific context are still too incomplete.

## Ask Before Expanding
Ask focused follow-up questions first when any of the following are unclear:
- what text is being edited,
- what section type it belongs to,
- what the intended meaning is,
- what the communication context is,
- whether surrounding text is needed to preserve meaning safely.

## Reporting Rule
If the input is insufficient, explicitly tell the user what information would improve editing accuracy.
