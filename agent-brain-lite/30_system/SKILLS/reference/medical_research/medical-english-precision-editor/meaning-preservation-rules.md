# Meaning Preservation Rules

## Core Rule
Editing must preserve the original factual meaning, evidence level, and intended scientific scope.

## Preserve
- study design implications
- evidence boundaries
- uncertainty level
- claim strength
- actual result meaning

## Important Rule
A cleaner sentence is not acceptable if it changes what the study supports.

## Reporting Rule
State where edits were made for precision without changing meaning.
