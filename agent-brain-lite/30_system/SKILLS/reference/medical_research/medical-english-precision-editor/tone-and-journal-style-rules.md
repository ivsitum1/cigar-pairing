# Tone and Journal Style Rules

## Core Rule
Journal-style refinement should improve professionalism without adding hype or false certainty.

## Good Tone Features
- direct but disciplined
- formal but natural
- precise without stiffness
- credible without overstatement

## Important Rule
Do not equate “international journal style” with inflated or overly ornate language.

## Reporting Rule
State when changes improve tone without changing claim strength.
