# Conclusion Boundary Rules

## Core Rule
The manuscript’s summary and conclusion language must remain anchored to the results actually shown.

## Common Problems
- conclusion stronger than the presented data
- abstract conclusion not matching main-text results
- discussion summary drifting beyond the figures/tables
- clinical or mechanistic wording exceeding the reported evidence

## Important Rule
A conclusion-result mismatch is not just a tone problem; it can become a major consistency risk.

## Reporting Rule
State when conclusion language should be narrowed to match the displayed evidence.
