# Version Drift Rules

## Core Rule
Many manuscript inconsistencies arise from revision-stage drift rather than from original study logic.

## Common Drift Types
- sample-size drift
- endpoint wording drift
- outcome-label drift
- analysis-description drift
- terminology drift
- revised abstract not matching revised results
- conclusion not updated after analysis changes

## Important Rule
Version drift is often a credibility problem even when each sentence looks plausible on its own.

## Reporting Rule
Name the specific drift pattern when present.
