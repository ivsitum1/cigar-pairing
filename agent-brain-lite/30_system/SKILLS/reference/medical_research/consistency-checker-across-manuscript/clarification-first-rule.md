# Clarification-First Rule

## Core Rule
Do not produce a long consistency review if the manuscript components under comparison are still too incomplete.

## Ask Before Expanding
Ask focused follow-up questions first when any of the following are unclear:
- which sections or components are being checked,
- whether the review is full-manuscript or targeted,
- whether figures/tables/supplements are available,
- whether the manuscript is in pre-submission or revision stage,
- whether the goal is contradiction detection or general alignment review.

## Reporting Rule
If the input is insufficient, explicitly tell the user what information would improve consistency-checking accuracy.
