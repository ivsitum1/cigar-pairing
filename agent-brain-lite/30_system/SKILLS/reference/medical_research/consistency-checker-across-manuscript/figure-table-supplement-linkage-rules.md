# Figure, Table, and Supplement Linkage Rules

## Core Rule
Display items and supplements must be correctly referenced and accurately described in the manuscript text.

## Check For
- figure-number mismatch
- table-number mismatch
- supplement citation mismatch
- display item cited but not explained correctly
- text claiming a result not actually shown in the cited display item

## Important Rule
A numbering issue may be minor, but a text-display mismatch can be a major credibility risk.

## Reporting Rule
Separate structural reference errors from deeper content mismatches.
