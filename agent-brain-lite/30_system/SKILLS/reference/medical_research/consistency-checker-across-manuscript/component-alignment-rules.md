# Component Alignment Rules

## Core Rule
The main manuscript components should be checked against each other, not in isolation.

## Compare Across
- title
- abstract
- methods
- results
- figures
- tables
- supplements

## Important Rule
A section can be individually well written and still be inconsistent with the rest of the manuscript.

## Reporting Rule
Make clear which sections are being compared in each major finding.
