# Clarification-First Rule

## Core Rule
Do not produce a long optimized title and abstract if the user has not provided enough study information to support accurate rewriting.

## Ask Before Expanding
Ask focused follow-up questions first when any of the following are unclear:
- what the study is about,
- what study design was used,
- what the main result is,
- what the study can and cannot claim,
- whether the user is providing a draft title, a draft abstract, or only a summary,
- whether a journal target or word constraint matters.

## Reporting Rule
If the input is insufficient, explicitly tell the user what information would improve optimization accuracy.
