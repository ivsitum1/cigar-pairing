# Clarification-First Rule

## Core Rule
Do not produce a long compliance review if the manuscript material is still too incomplete.

## Ask Before Expanding
Ask focused follow-up questions first when any of the following are unclear:
- what the study type is,
- what the manuscript type is,
- what guideline family is likely relevant,
- whether the review is for pre-submission or revision,
- which manuscript sections are available,
- whether the user has a checklist draft or only the manuscript text.

## Reporting Rule
If the input is insufficient, explicitly tell the user what information would improve compliance-checking accuracy.
