# Core Reporting Item Rules

## Core Rule
Focus first on high-risk reporting items that materially affect transparency and interpretability.

## Common High-Risk Items
- participant flow
- eligibility criteria
- sample/cohort description
- endpoint or outcome definition
- intervention or exposure definition
- analysis methods
- model development details
- model validation details
- missing data handling
- bias-control reporting
- subgroup / sensitivity analysis reporting
- registration or protocol reporting when relevant

## Important Rule
Do not let lower-value cosmetic items distract from missing core transparency elements.

## Reporting Rule
The compliance review should emphasize these items before secondary checklist details.
