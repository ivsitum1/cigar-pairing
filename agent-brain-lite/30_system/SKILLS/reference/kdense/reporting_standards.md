# Statistical reporting standards (APA-oriented)

## General

- Report exact p-values unless p &lt; .001.
- Always include effect size and 95% CI for primary analyses.
- Leading zero: use for statistics that can exceed 1 (t = 0.50); omit for p and r bounded by 1 (p = .032, r = .45).

## Templates

- **t-test:** t(df) = X.XX, p = .XXX, d = X.XX, 95% CI [low, high]
- **ANOVA:** F(df1, df2) = X.XX, p = .XXX, η² = .XX
- **Correlation:** r(df) = .XX, p = .XXX, 95% CI [low, high]
- **Chi-square:** χ²(df, N = n) = X.XX, p = .XXX
- **Regression:** β = X.XX, SE = X.XX, t = X.XX, p = .XXX (or OR with CI for logistic)

## Avoid

- "Not significant, therefore no effect."
- p-value alone without effect size.
- Changing tests after seeing data without labeling exploratory.

## Journal-specific

Confirm target journal (Vancouver vs APA vs ICMJE). Manuscript-facing text must not cite internal skill paths.
