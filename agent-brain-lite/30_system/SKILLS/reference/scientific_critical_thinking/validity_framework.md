# Scientific validity and evidence critique (condensed)

Adapted from K-Dense `scientific-critical-thinking` (MIT). Use with `SKILL_swiss-cheese`, `SKILL_grade-assessment`, `SKILL_literature-synthesis`.

## Study design

- Design matches question; causal claims require causal design or explicit emulation framing
- Comparison groups appropriate; attrition and selection bias noted

## Four validities

1. **Internal:** randomization, confounding control, selection, attrition  
2. **External:** sample, setting, generalizability to target population  
3. **Construct:** measurement validation, operational definitions  
4. **Statistical conclusion:** power, assumptions, test appropriateness

## Bias scan (quick)

- Performance, detection, attrition, reporting, funding/conflict  
- For reviews: RoB tool matched to design (Cochrane ROB2, ROBINS-I, QUADAS-2)

## GRADE linkage

- Start certainty; downgrade for risk of bias, inconsistency, indirectness, imprecision, publication bias  
- Do not upgrade observational evidence to "high" without exceptional justification
