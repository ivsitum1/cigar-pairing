# Scientific Writing Rules

## Purpose

This document establishes standards for writing scientific papers in accordance with PRISMA 2020 and GRADE approach. These rules ensure that all papers meet the highest standards of scientific reporting.

---

## PRISMA 2020 Compliance

### Title and Abstract

- [ ] Title identifies as systematic review and/or meta-analysis
- [ ] Structured abstract includes:
  - [ ] Objective
  - [ ] Data sources
  - [ ] Eligibility criteria
  - [ ] Methods (including statistical synthesis)
  - [ ] Results (summary of statistics, effect estimates, heterogeneity)
  - [ ] Conclusions

### Methods Section

#### Eligibility Criteria
- [ ] Population (P) clearly specified
- [ ] Intervention (I) clearly specified
- [ ] Comparison (C) clearly specified
- [ ] Outcomes (O) clearly specified (primary and secondary)
- [ ] Study design criteria specified

#### Information Sources
- [ ] Databases searched listed
- [ ] Search dates specified
- [ ] Other sources (hand search, experts, etc.) specified
- [ ] Grey literature sources specified

#### Search Strategy
- [ ] Complete search strategy for at least one database provided
- [ ] Search terms are reproducible
- [ ] Filters/limitations applied documented

#### Selection Process
- [ ] Number of reviewers specified
- [ ] Process described (independent screening, consensus)

#### Data Collection Process
- [ ] Number of reviewers for data extraction specified
- [ ] Process described (independent extraction, consensus)

#### Data Items
- [ ] Outcomes and summary measures specified
- [ ] Effect measures specified (RR, MD, OR, etc.)

#### Risk of Bias Assessment in Studies
- [ ] Tool used specified (e.g., Cochrane ROB 2)
- [ ] Number of reviewers specified
- [ ] How it is used in synthesis described

#### Effect Measures
- [ ] Effect measures clearly specified (RR, MD, OR, etc.)
- [ ] Rationale for choice explained

#### Synthesis Methods
- [ ] Statistical methods specified:
  - [ ] Models used (random effects, common effects)
  - [ ] Methods for heterogeneity assessment
  - [ ] Software used
- [ ] Subgroup analyses specified (if performed)
- [ ] Meta-regression specified (if performed)
- [ ] Sensitivity analyses specified
- [ ] Methods for publication bias assessment specified

### Results Section

#### Study Selection
- [ ] PRISMA flow diagram included (MANDATORY)
  - [ ] Records identified (databases, registries, other)
  - [ ] Records removed before screening
  - [ ] Records screened
  - [ ] Records excluded
  - [ ] Reports sought
  - [ ] Reports assessed
  - [ ] Studies included
  - [ ] Reasons for exclusion

#### Study Characteristics
- [ ] Table of included studies with:
  - [ ] Author, year
  - [ ] Study design
  - [ ] Participants (n, characteristics)
  - [ ] Interventions
  - [ ] Outcomes
  - [ ] Source of funding

#### Risk of Bias in Studies
- [ ] Summary of risk of bias assessment
- [ ] Per-outcome assessment (if applicable)
- [ ] Table or figure showing risk of bias

#### Results of Individual Studies
- [ ] For each study: effect estimates with confidence intervals
- [ ] Shown in forest plots or tables

#### Synthesis Results
- [ ] Forest plots for all outcomes
- [ ] Summary effect estimate with 95% CI
- [ ] Heterogeneity statistics (I², τ², Q-statistic with p-value)
- [ ] Prediction intervals (if applicable)
- [ ] Subgroup analysis results (if performed)
- [ ] Meta-regression results (if performed)
- [ ] Sensitivity analysis results

#### Reporting Biases
- [ ] Funnel plots (if ≥10 studies)
- [ ] Statistical tests (Egger's, Peters', etc.)
- [ ] Trim-and-fill results (if asymmetry detected)
- [ ] Interpretation of findings

#### Certainty of Evidence
- [ ] GRADE assessment for each outcome
- [ ] Summary of findings table

### Discussion Section

- [ ] Summary of evidence
- [ ] Discussion of heterogeneity
- [ ] Discussion of limitations:
  - [ ] Study-level limitations
  - [ ] Review-level limitations
- [ ] Conclusions:
  - [ ] Implications for practice
  - [ ] Implications for research

## GRADE Assessment Requirements

### For Each Outcome

#### Initial Quality
- [ ] Initial quality level specified:
  - [ ] RCTs: HIGH
  - [ ] Observational: LOW

#### Factors Decreasing Assessment
- [ ] Risk of bias (study limitations)
- [ ] Inconsistency (heterogeneity)
- [ ] Indirectness (PICO differences)
- [ ] Imprecision (wide CIs, few events)
- [ ] Publication bias

#### Factors Increasing Assessment
- [ ] Large effect
- [ ] Dose-response gradient
- [ ] All plausible confounders would reduce effect

#### Final Quality Rating
- [ ] HIGH (⊕⊕⊕⊕)
- [ ] MODERATE (⊕⊕⊕⊖)
- [ ] LOW (⊕⊕⊖⊖)
- [ ] VERY LOW (⊕⊖⊖⊖)

#### Summary of Findings Table
- [ ] Outcome name
- [ ] Expected absolute effects (intervention vs. control)
- [ ] Relative effect (95% CI)
- [ ] Number of participants (studies)
- [ ] Quality of evidence (GRADE rating)
- [ ] Comments/notes

## Statistical Reporting Standards

### Effect Estimates

- [ ] Point estimate reported
- [ ] 95% confidence interval reported
- [ ] Exact p-value reported (not just < 0.05)
- [ ] Both random and common effects reported (or rationale for one model)

### Heterogeneity

- [ ] I² with interpretation
- [ ] τ² (tau-squared)
- [ ] Q-statistic with degrees of freedom and p-value
- [ ] Interpretation of heterogeneity

### Tests

- [ ] Test for overall effect (z-statistic, p-value)
- [ ] Test for heterogeneity (Q-statistic, p-value)
- [ ] Test for subgroup differences (if applicable)
- [ ] Publication bias tests (if applicable)

### Sensitivity Analyses

- [ ] All sensitivity analyses reported
- [ ] Results compared with primary analysis
- [ ] Impact on conclusions discussed

## Key Principles

### Academic and Scientific Standards
- Always maintain scientific rigor and statistical accuracy
- Use academic language that sounds natural and human, not artificial
- Never fabricate or lie - all claims must be verifiable
- All statistical claims must be accurate and supported by evidence
- Cite sources appropriately using established academic formats
- Maintain objectivity and avoid overemphasizing findings

### Human Writing Characteristics (Agency-Based Approach)
**CRITICAL:** Human writing in academic work is NOT emotionality. It is:

**VISIBLE AGENCY:**
- Use "We did X", "Our protocol", "Our findings" instead of "The study found"
- Replace passive constructions with active voice showing ownership
- Specify who did what: "We performed sensitivity analyses" not "Sensitivity analyses confirmed"
- Add sample sizes and study counts: "across 10 studies with 831 participants"

**METHODOLOGICAL TRANSPARENCY:**
- Specify what was tested: "We performed sensitivity analyses across all outcome definitions"
- Explain decisions: "We included studies from 2014 onward per our PROSPERO protocol because..."
- Show process: "We assessed publication bias using Egger's test and found..."
- Always follow "we chose X" with "because..." or "to ensure..."

**OWNERSHIP OF UNCERTAINTIES:**
- Replace "It is unclear" with "Our data don't allow us to determine"
- Replace "High variability limits" with "Our data don't allow us to pinpoint the drivers"
- Replace "This finding requires careful interpretation" with "We interpret this finding with caution given [specific threat: publication bias p=0.0003, heterogeneity I²=44%]"
- Add consequences: "Consequently, we cannot determine whether..."

**ACCOUNTABILITY IN DECISIONS:**
- Show why decisions were made: "We chose this cutoff because..."
- Acknowledge trade-offs: "This approach prioritizes currency over comprehensiveness"
- Reference protocol when pre-registered: "per our PROSPERO protocol"
- Explain criteria, not just state them

**AVOID (Anti-Human Signals):**
- ❌ Performative emotions: "I'm frustrated", "This is frustrating", "I'm surprised"
- ❌ Subjective hedges: "I think", "I believe", "I'm not entirely sure", "unclear to me"
- ❌ Personal sentiments: These are WORSE than AI - they signal unprofessional writing

**Structural Variation (Subtle):**
- Allow some variation in paragraph structure
- Vary paragraph lengths
- Include occasional longer, more complex sentences
- Vary terminology slightly (within reason)

**Note:** The goal is to make the researcher VISIBLE through methodology and transparency, not through emotions. Agency-based writing shows ownership and accountability, which is what distinguishes human academic writing from AI-generated text.

### Surgical brevity (manuscript deliverables)

**Goal:** Kirurška točnost bez umjetne opsežnosti. Accuracy stays; verbosity drops.

| Rule | Action |
|------|--------|
| One job per sentence | Results/Discussion: one claim or finding per sentence |
| Verb > noun pile | Prefer "We analyzed…" over "The analysis of…" chains |
| Pseudo-commitment ban | `central`, `crucial`, `intricate`, `comprehensive`, `multifaceted` need data or go |
| Syntactic inflation cap | Non-Methods sentences >35 words → split or trim |
| Defensive hedging cap | Max one caution phrase per paragraph |
| No phantom denial | Do not negate claims the reader never made |
| Structure | Agency yes; PowerPoint subheadings in every paragraph no |

**Cross-ref:** `.cursor/rules/writing-avoid-ai.mdc` § Surgical brevity; `55_conversational_cognition.mdc` for chat vs manuscript channel split.

### Natural Academic Writing (Reducing AI Detection While Maintaining Academic Standards)

**CRITICAL:** These strategies maintain academic rigor, third-person/passive voice, and formal tone while making writing sound more natural and human.

#### 1. Reducing Mechanical Precision

**Problem:** Overly precise and technical word choice that prioritizes clarity to the point of sounding robotic.

**Solutions:**
- ✅ Mix technical precision with occasional natural phrasing: "The analysis demonstrated" → "Analysis showed" (occasionally)
- ✅ Use slightly less formal synonyms occasionally: "utilize" → "use", "facilitate" → "help", "demonstrate" → "show"
- ✅ Allow minor imprecision in non-critical statements: "approximately 10 studies" instead of always "exactly 10 studies"
- ✅ Use natural academic phrasing: "These findings suggest" instead of always "These findings demonstrate with statistical significance"
- ❌ Never sacrifice accuracy for naturalness in statistical claims or methods

**Examples:**
- ❌ "The statistical analysis demonstrated a significant association (p < 0.001)"
- ✅ "Analysis showed a significant association (p < 0.001)" (occasionally)
- ✅ "The statistical analysis demonstrated a significant association (p < 0.001)" (when precision is needed)

#### 2. Reducing Formulaic Organization

**Problem:** Overly structured text with predictable headings and sections that feel mechanical.

**Solutions:**
- ✅ Vary section introductions: Not every section needs "This section describes..."
- ✅ Occasionally start sections with findings or context instead of formal declarations
- ✅ Mix paragraph structures: Some paragraphs can start with data, others with interpretation
- ✅ Allow natural flow between related concepts without forcing rigid structure
- ✅ Use varied transition patterns (see below)
- ❌ Maintain clear organization and logical progression (don't sacrifice clarity)

**Examples:**
- ❌ "This section presents the results. The analysis included 10 studies..."
- ✅ "Analysis of 10 studies revealed..." (occasionally)
- ✅ "This section presents the results. The analysis included 10 studies..." (when structure is needed)

#### 3. Reducing Robotic Formality

**Problem:** Overly formal and polished writing that lacks variation and feels mechanical.

**Solutions:**
- ✅ Vary formality levels slightly within academic bounds: Mix "It was observed that" with "Observations showed"
- ✅ Use occasional direct statements: "The effect was significant" instead of always "It was observed that the effect demonstrated statistical significance"
- ✅ Allow natural academic phrasing: "Several studies reported" instead of always "A number of studies in the literature have reported"
- ✅ Reduce hedging when certainty is high: "The results indicate" instead of always "The results appear to indicate"
- ❌ Maintain academic tone and objectivity (never become casual or subjective)

**Examples:**
- ❌ "It was observed that the intervention demonstrated a statistically significant effect"
- ✅ "The intervention showed a significant effect" (occasionally, when appropriate)
- ✅ "It was observed that the intervention demonstrated a statistically significant effect" (when formality is needed)

#### 4. Maintaining Appropriate Impersonal Tone (Academic Standard)

**Note:** Impersonal tone is CORRECT for academic writing. The issue is when it becomes robotic, not that it's impersonal.

**Solutions:**
- ✅ Keep third-person and passive voice (academic standard)
- ✅ Add variation through sentence structure, not through personal pronouns
- ✅ Use agency-based phrasing: "The authors performed" or "We performed" (if journal allows) instead of only passive
- ✅ Vary passive constructions: "was performed", "was conducted", "was carried out"
- ❌ Never add first-person emotions or subjective statements

## Context Rule for Jargon vs. Clarity

These strategies are not contradictory — they apply to different sections:

- **Methods and Results**: prioritize technical precision. Use discipline-specific jargon.
  Maximizes perplexity (target >40) and accuracy. Specialist reviewers read these sections closely.
- **Introduction and Discussion**: prioritize comprehension. Use simpler language.
  Non-specialist reviewers and editors read these sections first; clarity aids acceptance.
  Target perplexity >30 in Discussion.

#### 5. Reducing Excessive Technical Jargon

**Problem:** Overuse of technical terms and complex sentence structures that feel unnatural.

**Solutions:**
- ✅ Mix technical terms with accessible language: "heterogeneity" → occasionally "variability between studies"
- ✅ Simplify complex sentences occasionally: Break very long sentences into two shorter ones
- ✅ Use technical terms consistently but allow natural phrasing around them
- ✅ Define terms naturally when first introduced, not formulaically
- ❌ Never sacrifice technical accuracy (use correct terms when needed)

**Examples:**
- ❌ "The meta-analysis demonstrated substantial between-study heterogeneity (I² = 75%)"
- ✅ "Studies varied substantially (I² = 75%)" (occasionally)
- ✅ "The meta-analysis demonstrated substantial between-study heterogeneity (I² = 75%)" (when precision is needed)

#### 6. Reducing Overly Formal Grammar

**Problem:** Grammatically perfect but overly formal structures that lack natural variation.

**Solutions:**
- ✅ Vary sentence beginnings: Not every sentence starts with "The" or "It was"
- ✅ Mix sentence types: Include occasional questions in discussion ("What might explain this finding?")
- ✅ Use varied clause structures: Some complex, some simpler
- ✅ Allow occasional sentence fragments in appropriate contexts (very sparingly, in discussion)
- ✅ Vary subordinate clause placement: Beginning, middle, end
- ❌ Maintain grammatical correctness (never use incorrect grammar)

**Examples:**
- ❌ "The analysis was performed. The results were obtained. The interpretation was conducted."
- ✅ "Analysis revealed significant effects. These findings suggest..." (varied structure)
- ✅ "The analysis was performed, revealing significant effects that suggest..." (varied clause placement)

#### 7. Natural Transitions (Not Mechanical)

**Problem:** Formulaic transitional phrases that feel robotic ("Furthermore", "Moreover", "Additionally" in predictable patterns).

**Solutions:**
- ✅ Vary transition words: Mix "Furthermore", "Moreover", "Additionally" with "Also", "In addition", "Similarly"
- ✅ Use content-based transitions: Connect ideas through meaning, not just transition words
- ✅ Occasionally omit transitions when connection is clear: Not every paragraph needs a transition word
- ✅ Use varied transition patterns: Some paragraphs can start with data, others with interpretation
- ✅ Mix formal and slightly less formal transitions: "Furthermore" → occasionally "Also" or "Similarly"

**Examples:**
- ❌ "The analysis was performed. Furthermore, the results were obtained. Additionally, the interpretation was conducted."
- ✅ "Analysis revealed significant effects. These findings align with previous research. The interpretation suggests..." (content-based transitions)
- ✅ "The analysis was performed. The results, which align with previous research, suggest..." (varied structure)

#### 8. Reducing Over-Sophisticated Clarity

**Problem:** Overly precise word choice that prioritizes sophistication to the point of sounding unnatural.

**Solutions:**
- ✅ Mix sophisticated and accessible language: "demonstrate" → occasionally "show", "utilize" → "use"
- ✅ Use natural academic phrasing: "The findings indicate" instead of always "The findings demonstrate with clarity"
- ✅ Allow occasional simpler phrasing when meaning is clear
- ✅ Vary vocabulary: Not always the most sophisticated word
- ❌ Never sacrifice clarity or accuracy

**Examples:**
- ❌ "The analysis demonstrated with statistical precision the significant association"
- ✅ "Analysis showed a significant association" (occasionally)
- ✅ "The analysis demonstrated a significant association" (when precision is needed)

#### 9. Adding Subtle Creativity (Within Academic Bounds)

**Problem:** Writing lacks the subtle creative variations that human writers naturally include.

**Solutions:**
- ✅ Vary sentence rhythm: Mix short and long sentences
- ✅ Use varied paragraph structures: Some analytical, some descriptive, some interpretive
- ✅ Include occasional rhetorical questions in discussion: "What might explain this discrepancy?"
- ✅ Vary emphasis: Some statements can be more direct, others more nuanced
- ✅ Allow natural academic voice: Not every statement needs to be perfectly balanced
- ❌ Never sacrifice academic standards or objectivity

**Examples:**
- ❌ "The results were significant. The interpretation was clear. The implications were important."
- ✅ "Results were significant. What might explain this finding? The interpretation suggests several possibilities." (varied structure and occasional question)

#### 10. Creative Grammar (Within Academic Standards)

**Problem:** Grammatically correct but predictable structures that lack natural variation.

**Solutions:**
- ✅ Vary clause structures: Some sentences with multiple clauses, some simpler
- ✅ Mix active and passive voice appropriately: Not always passive
- ✅ Vary sentence length: Some short (10-15 words), some longer (25-35 words)
- ✅ Use varied punctuation: Occasional semicolons, dashes for emphasis
- ✅ Allow natural academic flow: Not every sentence needs to be perfectly structured
- ❌ Never use incorrect grammar or informal language

**Examples:**
- ❌ "The analysis was performed. The results were obtained. The interpretation was conducted."
- ✅ "Analysis revealed significant effects—findings that suggest important clinical implications." (varied structure, dash for emphasis)

#### 11. Reducing Predictable Syntax

**Problem:** Limited syntax patterns favoring declarative sentences and repetitive structures.

**Solutions:**
- ✅ Vary sentence types: Mostly declarative, but occasional complex structures
- ✅ Mix sentence patterns: Subject-verb-object, but also varied structures
- ✅ Use varied clause orders: Main clause first, subordinate first, embedded clauses
- ✅ Include occasional longer, more complex sentences (25-35 words)
- ✅ Break patterns: Not every sentence follows the same structure
- ❌ Maintain clarity and grammatical correctness

**Examples:**
- ❌ "The study was conducted. The analysis was performed. The results were obtained."
- ✅ "Conducted across 10 studies, the analysis revealed significant effects that suggest..." (varied structure, participial phrase)

#### Implementation Strategy

**Balance is Key:**
- Use these strategies OCCASIONALLY, not always
- Maintain academic standards: Never sacrifice accuracy, clarity, or objectivity
- Keep third-person and passive voice (academic standard)
- Vary patterns naturally: Not every sentence needs variation
- Check with AI detectors: Verify that changes reduce AI score while maintaining quality

**Revision Workflow:**
1. Write first draft following academic standards
2. Check with AI detector
3. Identify sections with high AI scores
4. Apply these strategies to high-score sections
5. Re-check with AI detector
6. Ensure academic quality is maintained

### Statistical Best Practices
- Follow R best practices for meta-analysis
- Preserve reproducibility of analyses
- Clearly document all statistical decisions
- Always report exact p-values (not just p < 0.05)
- Always report effect estimates with 95% confidence intervals
- Distinguish between statistical and clinical significance
- Follow PRISMA 2020 and GRADE standards

### Medical Research Standards
- Follow CONSORT, STROBE, and PRISMA guidelines for reporting
- Consider Minimal Clinically Important Difference (MCID) when available
- Report Number Needed to Treat (NNT) for binary outcomes when appropriate
- Assess quality of evidence using GRADE approach
- Always consider both absolute and relative risks

### Reporting standards for clinical AI studies (SPIRIT-AI / CONSORT-AI)

For protocols or trial reports involving AI/ML interventions:

- **Protocol (SPIRIT-AI):** AI intervention description (input, output, interaction); data sources and selection; human–AI interaction specification; error analysis plan.
- **Trial report (CONSORT-AI):** AI version used; input data handling; human–AI interaction results; error analysis.

Complete standard SPIRIT/CONSORT first, then add SPIRIT-AI/CONSORT-AI extension items. Document model versioning and report subgroup performance. Full checklist and clinical AI documentation: `30_system/behavior_rules/18_ml_production.md` and `.cursor/rules/50_ml_mlops_standards.mdc`.

**AI-era transparency:** When AI tools are used in writing or analysis, declare their use and limitations where relevant (e.g. methods or acknowledgements), without duplicating the scope of `10_ai_writing_plagiarism.md`.
