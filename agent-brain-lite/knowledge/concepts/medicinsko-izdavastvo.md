# Medicinsko izdavaštvo — osnove

## Urednički tok (pojednostavljeno)

1. Predaja rukopisa (cover letter, prilozi, ORCID)
2. Desk review → peer review (često double-blind)
3. Odluka: prihvat / revizija / odbij
4. Proof i publikacija

## Često korišteni standardi

- ICMJE — autorstvo i sukob interesa
- CONSORT, STROBE, PRISMA — ovisno o dizajnu studije (detalji u parent brain)
- Vancouver stil referenci — za citiranje
