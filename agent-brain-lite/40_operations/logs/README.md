# Logs Directory

Use this directory for session-level and compact summaries.

## Recommended Files

- `session-[YYYY-MM-DD].md` for chronological notes
- `compact-[YYYY-MM-DD-HHMM].md` for context compaction snapshots
- `plan-[task-slug].md` for plan/execute/verify checkpoints

## Minimal Entry Format

- timestamp
- task
- action
- observation
- next step
